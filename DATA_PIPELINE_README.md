# Intelligent-Support-Ticket-Classification-with-RAG-DEPI-Project

## Milestone 1: Data Pipeline
This branch (`m1/data-pipeline`) contains the data collection, cleaning, and preprocessing pipeline for the support ticket dataset. 

**Key Deliverables:**
- `m1_data_pipeline.ipynb`: The fully executable Jupyter notebook containing the entire pipeline.
- `docs/eda_report.md`: Exploratory Data Analysis report detailing dataset characteristics.
- `docs/preprocessing_doc.md`: Documentation of the preprocessing and embedding steps.
- `data/processed/`: Contains the generated training, validation, and test splits along with precomputed sentence embeddings and EDA plots.
- `requirements.txt`: Python dependencies needed to reproduce the pipeline.
